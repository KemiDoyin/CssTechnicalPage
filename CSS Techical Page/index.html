Introduction

CSS is a language used to style a webpage. It stands for Cascading Style Sheets. It describes how html elements are to be displayed on screen and other media. It includes designs, layout, and dsiplay for different devices.

Border

CSS Borders allow to specify the width, style and color of an elemet's border. The Border-Style property specifies what kind of border style to show. Below are some of the properties and many more

  • Dotted - defines a dotted border
  • Dashed - defines a dashed border
  • Inset - gives a 3D inset border depending on the value given
  • Outset - gives a 3D outset border depending on the given value
h1.dotted {border-style: dotted;} h1.inset {border-style: inset;} h1.dashed {border-style: dashed;} h1.double {border-style: double;} h1.groove {border-style: groove;} h1.ridge {border-style: ridge;} Result:

Background

The background property is used to add Background effect for elements. It could be a background color or an image. You can set the color of a page to any color or you can decide to add an image instead. One can also decide to add opacity to the colors and image to give another effect.

body { background-color: chocolate; } div { background: rgba(231, 42, 8, 0.3) /* Red background with 30% opacity */ } body { background-image: url("paper.gif"); }
Margin

The CSS margin property is used to create space around elements, outside of the element border. The margins are set for each side of an element (top, right, bottom and left). That is

  • Margin-top
  • Margin-bottom
  • Margin-left
  • Margin-right
p { Margin-bottom: 25px; margin-top: 20px; margin-left: 45px; margin-right: 10px; } Result:
